## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/shared_element.git
```

Go to the project directory

```bash
  cd shared_element
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
